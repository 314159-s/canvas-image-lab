# canvas-image-lab

一个用来练手的 React + Vite + Canvas 图片处理小项目。

## 功能

- 上传 JPG / PNG 图片
- 使用 Canvas 预览图片
- 查看原图 / 效果图
- 调整亮度和对比度
- 灰度化和反色处理
- 使用预设效果
- 下载处理后的 PNG 图片

## 运行项目

```powershell
npm.cmd install
npm.cmd run dev
```

启动后访问：

```text
http://127.0.0.1:5173/
```
